# Como hospedar seu site GRÁTIS (para usar no Amazon Associados)
# ==============================================================

## OPÇÃO 1 — GitHub Pages (GRÁTIS, mais recomendado) ⭐

### Passo a passo:
1. Acesse: https://github.com e crie uma conta (grátis)
2. Clique em "New repository" (novo repositório)
3. Nome do repositório: fitcomestilo
4. Marque "Public"
5. Clique "Create repository"
6. Clique "uploading an existing file"
7. Arraste o arquivo index.html (da pasta site/) para lá
8. Clique "Commit changes"
9. Vá em Settings → Pages
10. Em "Source", selecione "main" e clique "Save"
11. Seu site vai ficar em: https://SEUUSUARIO.github.io/fitcomestilo

### O que colocar no Amazon Associados:
   https://SEUUSUARIO.github.io/fitcomestilo

---

## OPÇÃO 2 — Netlify (GRÁTIS, mais fácil)

### Passo a passo:
1. Acesse: https://www.netlify.com e crie conta (grátis)
2. Na tela inicial, arraste a PASTA "site" inteira para o Netlify
3. Pronto! Ele gera um link tipo: https://fitcomestilo-abc123.netlify.app
4. Você pode personalizar o nome em Site Settings → Change site name

### O que colocar no Amazon Associados:
   https://fitcomestilo.netlify.app

---

## OPÇÃO 3 — Vercel (GRÁTIS)

### Passo a passo:
1. Acesse: https://vercel.com e crie conta (grátis com GitHub)
2. Importe o repositório do GitHub (se usou a Opção 1)
3. Ou faça upload direto
4. Link gerado: https://fitcomestilo.vercel.app

---

## OPÇÃO 4 — Só usar seu Instagram/Facebook (mais simples, mas limitado)

No campo "Insira seu site" do Amazon, você pode colocar:
   https://www.instagram.com/fitcomestilo
   ou
   https://www.facebook.com/fitcomestilo

⚠️ Funciona para se cadastrar, mas ter um site próprio passa mais credibilidade.

---

## RESUMO: O que colocar no Amazon Associados AGORA

### Campo "Insira seu site":
   → Se já hospedou: cole a URL do seu site (GitHub Pages, Netlify ou Vercel)
   → Se ainda não hospedou: cole https://www.instagram.com/fitcomestilo

### Campo "Insira o URL do seu app":
   → Deixe em branco (você não tem app)
